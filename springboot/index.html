@SpringBootTest
Starts a web environment for unitTesting. Initializes a rest template for http-requests to the controllers.
@DataJpaTest
Initializes a H2 database, Hibernate, SpringData and the DataSource.
@RunWith(SpringRunner.class)

Sets the runner for the test class.

Apparently, this is not sufficient for Controllers where @TestConfiguration leads to a cyclic dependecy error.

@TestConfiguration

  • @MockBeans from the surrounding class are usable in the TestConfiguration.
